# node-wiki

Simple editable Markdown wiki built with node using `fs`

## Setup

Clone this repo and `cd` into the cloned directory

```
$ git clone <repo-url>
$ cd <directory>
```

Install dependencies

```
$ npm i
```

Start the local server

```
$ gulp start
```

That's it! Vist `http://localhost:8080` in your browser.

## Tech

- Node.js
- Handlebars
- Markdown
- BassCSS
