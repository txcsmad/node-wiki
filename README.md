# node-wiki

Simple editable wiki built with node using `fs`

## Setup

Clone this repo and `cd` into it

```
$ git clone <repo-url>
$ cd <repo-directory>
```

Install dependencies

```
$ npm i
```

Start the local server

```
$ gulp start
```

That's it! Vist `http://localhost:8080` in your browser.

## Tech

- Node.js
- Handlebars
- BassCSS
